# Mentee Management System
A single page application which can be used to manage mentees or students.


## Technologies Used


 - Javscript
 - HTML
 - CSS
 
 ## Features
 
 
 
 - Add/Delete mentees.
 - Rate them.
 - Add comments.
 - Edit rating and comments.
 - Sort by Rating.
 - Localstorage used for data persistence.
